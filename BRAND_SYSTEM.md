# Line Bible — Luxury Brand System

## Brand Position
Line Bible is not a generic odds site.
It is a luxury market terminal for serious sports bettors.

Keywords:
- precise
- intelligent
- composed
- elite
- nocturnal
- instrument-grade
- private-club premium

## Core Visual Metaphor
A night-flight cockpit for sports trading.
Not literal aviation cosplay.
A premium command center with illuminated instruments, signal lights, layered glass, and confidence.

## Color System
### Core neutrals
- Obsidian: `#06080D`
- Midnight: `#0A0E14`
- Panel: `#0F141C`
- Surface: `#131A24`
- Divider: `rgba(205,216,235,0.06)`
- Text primary: `#EDF2FB`
- Text secondary: `#97A4BC`

### Signature accents
- Champagne Gold: `#D8BA76`
- Deep Gold: `#B69558`
- Avionics Blue: `#7FA6FF`
- Signal Green: `#24C997`
- Warning Red: `#F06F82`

## UI Rules
- Use gold sparingly for prestige and selected states
- Use green for active/live/positive only
- Use blue for informational overlays and chart glow
- Keep red rare and meaningful
- Prefer tables, modules, and rails over giant marketing cards
- Every panel should feel like an instrument module

## Typography Direction
- Clean sans serif for UI
- Tight tracking on large headings
- Tabular-feeling number treatment for odds, ROI, CLV, P&L
- Small uppercase labels for instrument captions

## Art Direction
### Imagery
- Night cockpit glow
- Precision controls
- Illuminated instruments
- Reflections on dark glass
- Premium metallic warmth

### Avoid
- cartoon plane imagery
- casino neon overload
- loud gradients everywhere
- childish sports clichés

- subtle panel edge lighting
- illuminated status dots
